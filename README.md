<h2>• EVENTS MANAGEMENT SYSTEM</h2>
<br>
▪ Designed and developed a web application using Core and Advanced Java for the back-end, and HTML, CSS, and Bootstrap for the front-end. MySQL was used as the database.
<br>
▪ In this project, users can book various events, send enquiries or feedback to the administrator, and check thestatus of their bookings. The administrator has access to approve bookings, as well as add or delete events from the list.
<br>
▪ The project incorporates concepts such as JDBC, Servlets, and JSP.
